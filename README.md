# Git Advanced Demo

## Overview

This repository was created as part of a DevOps learning program to practice advanced Git operations, GitHub collaboration workflows, CI/CD concepts, and GitHub Actions automation.

The project demonstrates hands-on experience with branching strategies, merge conflict resolution, rebasing, stashing, pull requests, and workflow automation.

---

## Objectives

* Understand Git branching workflows
* Practice merge conflict resolution
* Learn Git Rebase
* Use Git Stash for temporary changes
* Create and merge Pull Requests
* Understand CI/CD concepts
* Implement GitHub Actions workflows
* Explore Docker integration concepts within CI/CD pipelines

## GitHub Actions Workflow

The repository contains a GitHub Actions workflow that:

* Triggers on push
* Checks out repository code
* Displays repository information
* Displays current date
* Displays Git version

Workflow location:

```text
.github/workflows/basic.yml
```
